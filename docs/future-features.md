# Kitsune - Future Features (v0.1)

## Purpose

Dokumen ini berisi seluruh fitur masa depan Kitsune.

Fitur dalam dokumen ini TIDAK termasuk MVP.

AI Agent tidak boleh mengimplementasikan fitur di dokumen ini kecuali ada instruksi eksplisit.

---

# Philosophy

Future Features digunakan untuk:

- Menyimpan ide proyek
- Menentukan arah jangka panjang
- Mencegah fitur masa depan masuk ke MVP

---

# Current MVP Scope

MVP hanya mencakup:

```text
Comic Reader
Bookmarks
Playlists
Settings
Local Library
```

Semua fitur di bawah ini berada di luar scope MVP.
